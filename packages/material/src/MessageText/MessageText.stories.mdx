# MessageText

The MessageText component used to display text of chat message. Should be wrapped into MessageBox to display for user.

<Canvas>
	<Story name="Overview">
		<MessageBox author={{ username: 'Hello World' }} position="single" orientation="left" component={MessageText}>
			That's a message text wrapped into MessageBox component.
		</MessageBox>
	</Story>
</Canvas>

MessageText is a simple view component that can be used alone.

export const Template = (args) => <MessageText {...args} />;

<Canvas>
	<Story
		args={{
			children: "Hey! I'm a standalone message text!",
		}}
		name="Standalone"
	>
		{Template.bind({})}
	</Story>
</Canvas>

You can also change the color of message:

<Canvas>
	<Story args={{ color: 'primary', children: 'I am colored!' }} name="Colored">
		{Template.bind({})}
	</Story>
</Canvas>
